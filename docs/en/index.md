# SOPRAN English Docs

This is the English landing page for SOPRAN public documentation.

The package guides already expose bilingual `GuidePage` objects through
`language="ja"` and `language="en"`. The public MkDocs site uses
`Lang: 日本語/English` in the header so the same language-switching label is
available in published docs, package guides, and notebook displays.

Full page-by-page English navigation is still being expanded. Until each page
has a dedicated translation, the Japanese documentation and the package-level
English guides are the authoritative fallback.
